Тестовое задание
===============================

Инструкция по установке:
-------------------

```
1) Перейти в директорию домена, на котором будет тестироваться задание
2) Выполнить команду git clone git@github.com:zergbz1988/test2.git, либо скачать архив и распаковать в папку домена
3) Импортировать данные в БД с помощью файла db.sql (в корне проекта)
4) Данные БД: host: localhost name: test2; login: root; password: ''
5) Для входа Администратором: login: admin@admin.com, password: 123qwe
6) Заходить по адресу: http://ВАШ ДОМЕН/
7) Если не логиниться доступны 3 страницы: 
    - Одобренные отзывы с формой отправки нового
    - Все отзывы
    - Страница входа
   После авторизации становится доступна страница "Проверка отзывов", появляется кнопка "Выход"
```
